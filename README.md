SmartFit AI is an AI-powered gym and fitness assistant developed as part of an
internship-oriented project. The system integrates machine learning, computer

---

## Project Overview

The goal of this project is to design a modular fitness platform that can:
- Track workouts using pose estimation
- Analyze user performance
- Recommend diets and gyms
- Predict habits and fitness consistency
- Simulate smart gym IoT features
- Provide a virtual gym assistant

The project is designed with a clean separation between backend logic and frontend UI.

---

## Features

- Workout detection and repetition counting using computer vision
- Performance score calculation based on workout data
- AI-based diet recommendation
- Habit tracking and prediction using machine learning
- Gym recommendation system based on location and workout type
- Virtual gym buddy assistant
- Smart gym IoT simulation (heart rate, calories, steps)
- REST APIs for all backend services
